1. Translate (Move Elements)

The translate functions move elements from their original position.

translateY()

translate()

translate3d()

3D

6. Combined Transforms

You can combine multiple transform functions by separating them with spaces.

7. Transform Origin

The transform-origin property sets the point around which transformations occur.

top left

8. Transform Style

The transform-style property controls how child elements are rendered in 3D space.

preserve-3d

transform-style: preserve-3d;

Preserves 3D positioning of child elements

9. Perspective

The perspective property adds depth to 3D transformations.

With Perspective

3D
perspective: 500px;

Creates depth for 3D transforms
