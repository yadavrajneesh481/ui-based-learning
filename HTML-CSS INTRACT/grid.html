Understanding: grid-template-columns: repeat(2, 1fr)

üîç Breaking Down the Code

grid-template-columns: repeat(2, 1fr);

grid-template-columns = Defines column widths

repeat() = Function to repeat a pattern

2 = Number of columns to create

1fr = Each column gets 1 fraction of available space

üìä What "1fr" Means

"fr" = fraction unit
‚Ä¢ 1fr = 1 part of the available space
‚Ä¢ 2fr = 2 parts of the available space
‚Ä¢ If you have 1fr + 1fr = each gets 50% of space
‚Ä¢ If you have 1fr + 2fr = first gets 33%, second gets 67%
