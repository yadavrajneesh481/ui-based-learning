üìñ Negative Z-Index Example

Elements with negative z-index go behind their parent container:

This text has z-index: 1

üîç Key Rules:
  • Only works on positioned elements (position: relative, absolute, fixed, sticky)
  • Higher z-index = appears on top
  • Default z-index is usually 0
  • Negative z-index sends elements behind their parent
  • Elements with the same z-index stack based on HTML order (last one on top)
