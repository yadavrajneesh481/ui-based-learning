üìå Day 8: Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets. Think of it like this: HTML is the skeleton of a webpage (structure), and CSS is the clothes and makeup (styling). CSS tells the browser HOW to display HTML elements - colors, sizes, positions, fonts, etc.
üéØ Key Points to Remember:
  • CSS makes websites look beautiful and organized
  • Without CSS, websites look plain and boring (like plain text)
  • CSS is separate from HTML (though it can be mixed)
  • One CSS file can style multiple HTML pages
1Ô∏è‚É£ Inline CSS (‡§∏‡•Ä‡§ß‡•á HTML element ‡§Æ‡•á‡§Ç)

CSS directly written in the HTML tag itself.

Hinglish: Yeh tarika sirf ek element ko style karne ke liye use hota hai. Agar 100 paragraphs hain to 100 baar likhna padega!

2Ô∏è‚É£ Internal CSS (HTML file ke andar)

CSS written inside <style> tag in the <head> section.

<head> <style> p { color: blue; } </style> </head>

Hinglish: Ek hi file mein HTML aur CSS dono hote hain. Chhote projects ke liye theek hai.

3Ô∏è‚É£ External CSS (Alag file - ‡§∏‡§¨‡§∏‡•á ‡§Ö‡§ö‡•ç‡§õ‡§æ ‡§§‡§∞‡•Ä‡§ï‡§æ)

CSS in a separate file (style.css) linked to HTML.

<!-- In HTML file --> <link rel="stylesheet" href="style.css"> <!-- In style.css file --> p { color: green; }

Hinglish: Yeh sabse professional tarika hai! Ek CSS file se multiple HTML pages ko style kar sakte ho. Maintenance bhi easy hai.

1. Universal Selector (*)

Selects ALL elements on the page.

* { margin: 0; padding: 0; }

Real-world example: Jab aap sabhi elements se default spacing hatana chahte ho.

2. Element Selector

Selects all elements of a specific type.

p { color: red; } h1 { font-size: 30px; }

Real-world example: Sabhi paragraphs ko red color dena.

3. Class Selector (.classname)

Selects elements with a specific class. One element can have multiple classes.

.button { background: blue; padding: 10px; } <button class="button">Click Me</button>

Real-world example: Multiple buttons, cards, or containers ko same style dena.

4. ID Selector (#idname)

Selects a unique element with a specific ID. Each ID should be used only ONCE.

#header { background: navy; color: white; } <div id="header">Welcome</div>

Real-world example: Navbar, footer, main content area - unique sections ke liye.

5. Grouping Selector (,)

Apply same style to multiple selectors.

h1, h2, h3 { color: purple; font-weight: bold; }

Real-world example: Sabhi headings ko same color dena bina ek-ek karke likhe.

Q1: CSS kya hota hai aur HTML se alag kyu hota hai?
Answer:
HTML structure provide karta hai (skeleton), CSS styling provide karta hai (clothes).

Answer:
Inline: Style attribute directly HTML tag mein. Sirf ek element ke liye. Reuse nahi kar sakte.
Internal: <style> tag HTML file ke head mein. Ek file mein CSS aur HTML. Chhote projects ke liye theek.
External: Alag CSS file. Multiple HTML pages ko style kar sakte ho. Professional aur maintainable.

Production websites mein hamesha external CSS use hota hai kyunki scalable aur organized hota hai.
Q3: Class selector aur ID selector mein kya difference hai?
Answer:
Class (.classname): Multiple elements ke liye. Ek element mein multiple classes ho sakte hain.
ID (#idname): Unique element ke liye. Ek page mein ek ID sirf ek baar use hona chahiye.

Example:

Class = reusable, ID = unique. Agar 10 buttons hain to class use karo, agar sirf ek navbar hai to ID use karo.

- Reordering without changing HTML
- Better browser support

justify-content: Main axis (default: horizontal) ‡§™‡§∞ alignment control ‡§ï‡§∞‡§§‡§æ ‡§π‡•à‡•§
