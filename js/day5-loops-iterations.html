For Loop - Complete Deep Dive

For loop JavaScript mein sabse commonly used iteration structure hai. Yeh tab use karte hain jab humein pata ho ki exactly kitni baar loop chalana hai. For loop mein teen parts hote hain: initialization, condition, aur increment/decrement.

For Loop Anatomy aur Working:

üîÑ Loop Structure Breakdown:

for (initialization; condition; increment/decrement) {
  // Loop body - code to execute
}

// Example:
for (let i = 0; i < 5; i++) {
  console.log("Iteration:", i);
}
  • Initialization: Loop variable set karna (let i = 0)
  • Condition: Kab tak loop chalana hai (i < 5)
  • Increment: Har iteration ke baad variable change (i++)
  • Execution Order: Init ‚Üí Condition ‚Üí Body ‚Üí Increment ‚Üí Condition...

For Loop Variations aur Use Cases:

Interview Questions

1. Basic: for, while, aur do-while loop mein kya difference hai?

Answer: for loop fixed iterations ke liye, while condition check karke chalti hai, do-while pehle execute phir check karti hai.

2. Basic: for...of aur for...in mein kya difference hai?

Answer: for...of values iterate karta hai (arrays ke liye), for...in keys/indices iterate karta hai (objects ke liye).

3. Intermediate: break aur continue statement ka use kab karte hain?

Answer: break loop se completely bahar nikalne ke liye, continue current iteration skip karke next iteration pe jaane ke liye.

4. Intermediate: Infinite loop kya hai aur kaise avoid karte hain?

Answer: Loop jo kabhi end nahi hoti. Condition properly set karo, counter increment/decrement karo, break statements use karo.

5. Advanced: Nested loops ki time complexity kya hoti hai?

Answer: O(n¬≤) for two nested loops, O(n¬≥) for three nested loops. Performance impact hota hai large datasets pe.

6. Advanced: Label statements kya hain aur kab use karte hain?

Answer: Nested loops mein specific loop se break/continue karne ke liye. outer: for(...) { inner: for(...) { break outer; } }
