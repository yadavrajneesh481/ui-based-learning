Conditional statements programming mein decision making ke liye use hote hain. Yeh program ko different paths pe le jaate hain based on certain conditions. JavaScript mein if-else statements boolean expressions evaluate karte hain aur code execution control karte hain.

Interview Questions

1. Basic: If-else aur switch statement mein kya difference hai?

Answer: If-else ranges aur complex conditions ke liye, switch exact values ke liye better hai. Switch faster hai multiple conditions ke liye.

2. Basic: Ternary operator kya hai? Syntax kya hai?

Answer: Short if-else statement. Syntax: condition ? trueValue : falseValue

3. Intermediate: JavaScript mein falsy values kaun si hain?

Answer: false, 0, "", null, undefined, NaN - yeh 6 values falsy hain, baaki sab truthy.

4. Intermediate: [] aur {} truthy hain ya falsy?

Answer: Dono truthy hain, kyunki yeh objects hain. Empty bhi ho to truthy rahte hain.

5. Advanced: Switch statement mein break ka kya role hai?

Answer: Break nahi lagaoge to fall-through hoga - next cases bhi execute honge. Intentional fall-through sometimes useful hai.
