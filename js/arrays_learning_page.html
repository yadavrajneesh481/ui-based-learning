üìö Arrays Kya Hain?

Arrays ek special type ka variable hai jo multiple values ko ek saath store kar sakta hai. Socho agar tumhe 100 students ke names store karne hain - kya tum 100 alag alag variables banaaoge? Nahi na! Yahan arrays kaam aate hain. Arrays JavaScript mein objects hain, primitive values nahi!

Definition (Technical):

An array is an ordered collection of values that can store multiple data items of any type in a single variable. Each item in an array has a numeric position known as its index (starting from 0). Arrays are reference types and inherit from Array.prototype.

üß† Array Deep Concepts:

  • Zero-Indexed: Pehla element index 0 pe hota hai, 1 pe nahi
  • Dynamic Size: Runtime pe size badha ya ghata sakte hain
  • Mixed Types: Ek array mein different data types store kar sakte hain
  • Reference Type: Arrays memory mein reference store karte hain
  • Sparse Arrays: Missing indices undefined hote hain
  • Array-like Objects: Length property aur numeric indices wale objects

‚úèÔ∏è Arrays Ko Modify Karna

Arrays ko modify karne ke liye JavaScript mein kaafi saare methods hain. Chaliye sabse important methods ko detail mein samajhte hain.

push() - End mein add karo

push() method array ke end mein nayi value add karta hai aur naye array ki length return karta hai.
let fruits = ["Apple", "Banana"]; fruits.push("Mango"); // ["Apple", "Banana", "Mango"]

pop() - End se remove karo

pop() method array ke end se last element remove karta hai aur us element ko return karta hai.
let fruits = ["Apple", "Banana", "Mango"]; let removed = fruits.pop(); // "Mango" // fruits = ["Apple", "Banana"]

shift() - Start se remove karo

shift() method array ke beginning se first element remove karta hai.
let fruits = ["Apple", "Banana", "Mango"]; let removed = fruits.shift(); // "Apple" // fruits = ["Banana", "Mango"]

unshift() - Start mein add karo

unshift() method array ke beginning mein nayi value add karta hai.
let fruits = ["Banana", "Mango"]; fruits.unshift("Apple"); // ["Apple", "Banana", "Mango"]

splice() - Kahi bhi add/remove karo

splice() sabse powerful method hai - tum kisi bhi position se elements add ya remove kar sakte ho. Syntax: splice(startIndex, deleteCount, item1, item2, ...)
let fruits = ["Apple", "Banana", "Mango", "Orange"]; // Position 1 se 2 elements remove karo fruits.splice(1, 2); // ["Apple", "Orange"] // Position 1 par naye elements add karo fruits.splice(1, 0, "Grapes", "Kiwi"); // ["Apple", "Grapes", "Kiwi", "Orange"]

üîÑ Arrays Ko Loop Karna (Iteration)

Array ke har element ko access karne ke liye hum loops use karte hain. JavaScript mein arrays ko iterate karne ke kaafi tarike hain.

for loop - Traditional Way

let fruits = ["Apple", "Banana", "Mango"]; for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); }

forEach() - Modern & Clean

forEach() method har element ke liye ek callback function run karta hai. Ye sabse commonly used method hai.
let fruits = ["Apple", "Banana", "Mango"]; fruits.forEach(function(fruit, index) { console.log(index + ": " + fruit); }); // Arrow function ke saath fruits.forEach((fruit, index) => { console.log(`${index}: ${fruit}`); });

for...of - Simple Iteration

let fruits = ["Apple", "Banana", "Mango"]; for (let fruit of fruits) { console.log(fruit); }

üéì Interview Questions

Q1: What is the difference between push() and unshift() methods?

Answer: push() adds elements at the end of the array, while unshift() adds elements at the beginning of the array. push() is faster because it doesn't need to reindex all elements, whereas unshift() has to shift all existing elements to make room at the start.

Hinglish: push() array ke end mein add karta hai aur unshift() starting mein. push() fast hai kyunki wo baki elements ko shift nahi karta, lekin unshift() ko saare elements ko aage shift karna padta hai.

Q2: How do you find the length of an array?

Answer: Use the length property: arrayName.length. It returns the number of elements in the array. The length is always one more than the highest index.

Example: let arr = [1, 2, 3]; console.log(arr.length); // 3

Q3: What will splice(2, 1, 'x', 'y') do to an array?

Answer: It will start at index 2, remove 1 element, and insert 'x' and 'y' at that position.

Example:
let arr = ['a', 'b', 'c', 'd'];
arr.splice(2, 1, 'x', 'y');
// Result: ['a', 'b', 'x', 'y', 'd']

Hinglish: Index 2 se start karke 1 element remove karega ('c') aur uski jagah 'x' aur 'y' insert kar dega.

Q4: What's the difference between forEach() and map()?

Answer: forEach() iterates through an array but doesn't return anything (returns undefined). map() creates and returns a new array with the results of calling a function on every element.

Hinglish: forEach() sirf loop karta hai aur kuch return nahi karta. map() ek naya array banata hai jismein har element par function call karke results store hote hain.

Q5: How can you remove a specific element from an array by value (not index)?

Answer: First find the index using indexOf() or findIndex(), then use splice() to remove it. Or use filter() to create a new array without that element.

Q6: Can an array in JavaScript hold different data types?

Answer: Yes! JavaScript arrays can hold any combination of data types - numbers, strings, booleans, objects, even other arrays.

Hinglish: Haan bilkul! JavaScript mein ek hi array mein numbers, strings, booleans, objects, sab kuch store kar sakte ho.

Q7: What happens when you access an index that doesn't exist in an array?

Answer: It returns undefined. JavaScript doesn't throw an error for accessing non-existent indices.

Hinglish: Agar tum kisi aisi index ko access karo jo exist nahi karti, to JavaScript undefined return karega, error nahi dega.
